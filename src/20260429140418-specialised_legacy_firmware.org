Used by factions as a reward for mission contracts. Known to be
salvaged from signal sources. Known to be found in small settlement
data point networks.

Specialised legacy firmware is the control software from earlier
versions of the technology. These can be rare finds and have some
uses in developing new versions with backwards compatibility.
